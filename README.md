# SwingPainter
一个本来想做成像Windows Xp风格的画图板。（其实只是个大作业）

![预览](img/preview.png)

# feature
目前完成的功能:
 - 直线，橡皮，椭圆，笔刷，矩形，油漆桶
 - 保存图像为jpg。
 - 打开图像。
 - 撤销，重做。
 - 选择颜色。
